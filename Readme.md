Videos for demo presentations
--------

This is a collection of open source or public domain movies that can be used during your demonstrations.
All videos have been encoded in MP4 format (H264 video / AAC audio). You may also check [the internet archive][internetarchive] for more media.
To start streaming the media use node streamer.js and set a video HTML5 tag to show http://localhost:1982/videoname.mp4.
